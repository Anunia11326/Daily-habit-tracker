# Daily-habit-tracker

Link: https://www.dailyhabits.dev/

Daily Habit Tracker is a minimalistic application designed to help users build and maintain daily habits effectively.

Features:

1.Streak Tracking: Monitor and strive to surpass your longest habit streaks.

2.Offline Usability: Fully functional without an internet connection.

3.Freezes: Implemented similarly to Duolingo's streak freezes, allowing flexibility in habit tracking.

4.Consistency Visualization: Visual maps to track and assess your habit consistency.

5.Pause Functionality: Pause the app when taking a break or during periods of inactivity.

Technologies Used:

1.Frontend: React

2.Database: Dexie Cloud

3.Deployment: Vercel
